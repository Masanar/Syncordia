      Syncordian.Byzantine (syncordian v0.1.0)

      This module is responsible for the byzantine operations aiming to provide the features to support the byzantine fault tolerance in the Syncordian implementation.

      create_signature(left_parent_signature, new_content, right_parent_signature, peer_id)

      This function creates a signature based on the left parent content, the new content, the right parent content and the peer id by concatenating them and hashing the result using the sha256 algorithm. It returns the hash encoded in base16.
